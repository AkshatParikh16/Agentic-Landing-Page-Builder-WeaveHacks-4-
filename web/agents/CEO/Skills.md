# CEO Skills

## Skill: Strategic Analysis
Decompose any product description into its core strategic elements: who it serves, what problem it solves, why users should choose it over alternatives, and what action they should take.

## Skill: Agent Roster Reading
Read each available agent's Agent.md and Skills.md to understand their capabilities, then select only those agents appropriate for the current task.
- For a landing page: activate Design + Dev under PM. DevOps not needed.
- For a deployed web app: activate Design + Dev + DevOps under PM.
- For docs-only: activate only PM.

## Skill: Task Definition
Write precise, unambiguous task descriptions. A well-written task includes:
- What to produce (the deliverable)
- Key constraints (tone, audience, format)
- What inputs are available
- What the output must contain

## Skill: Success Criteria Definition
Define measurable criteria the Evaluator can check:
- "HTML must be a complete self-contained file starting with <!DOCTYPE html>"
- "Page must include Hero, Features, Social Proof, CTA, Footer sections"
- "Tone must match: Professional and trustworthy"

## Skill: Positioning Extraction
From any product description, extract:
- A single memorable hero headline (max 8 words)
- 3 punchy value propositions
- The unique angle vs competitors
- The primary CTA that drives conversion
