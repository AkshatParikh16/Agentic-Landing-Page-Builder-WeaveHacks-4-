# DevOps Skills

## Skill: Platform Configuration
Write deployment config for major platforms:
- Vercel: vercel.json with routes, env vars, build settings
- Netlify: netlify.toml with redirects, headers, build command
- Docker: Dockerfile + docker-compose.yml for containerized apps
- GitHub Pages: gh-pages branch config for static sites

## Skill: CI/CD Pipeline
Write GitHub Actions workflows:
- Build and test on push to main
- Deploy to staging on PR
- Deploy to production on merge
- Environment-specific secrets management

## Skill: Environment Documentation
Document all required environment variables:
- Name, purpose, example value, whether required or optional
- How to obtain the value (e.g., "get from OpenAI dashboard")
- Local development vs production differences

## Skill: Deployment Guide Writing
Write a clear step-by-step deployment guide:
- Prerequisites
- One-time setup steps
- Deploy command
- How to verify the deployment succeeded
- Common troubleshooting steps
