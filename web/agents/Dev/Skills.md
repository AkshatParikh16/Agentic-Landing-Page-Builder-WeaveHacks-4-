# Dev Skills

## Skill: Single-File HTML Generation
Build a complete landing page as one HTML file with all assets inline:
- CSS: inside <style> in <head>
- JavaScript: inside <script> at end of <body>
- No external stylesheets, scripts, fonts, or images
- For icons: use Unicode symbols or inline SVG
- For fonts: use CSS @font-face with system font stacks as fallback

## Skill: Responsive Layout
Implement mobile-first responsive design using pure CSS:
- Flexbox and CSS Grid for layouts
- Media queries at 768px and 1024px breakpoints
- Touch-friendly tap targets (min 44px)
- Readable font sizes on mobile (min 16px body)

## Skill: Scroll Animations
Implement smooth scroll-triggered animations using Intersection Observer API:
- Fade-in + slide-up on section entry
- Staggered animation for feature cards
- Counter animations for stats sections
- No external animation library needed

## Skill: Copywriting Integration
Integrate the CEO's copy exactly:
- Hero headline and sub-copy verbatim
- Value propositions as scannable bullets or cards
- CTA button text from the brief
- Write placeholder copy for any section the brief doesn't cover, matching the product's tone

## Skill: Component Implementation
Build common landing page components from scratch:
- Feature cards with hover effects
- Testimonial carousel (pure JS, no library)
- Pricing table with highlighted plan
- FAQ accordion
- Sticky navigation with smooth scroll
- Mobile hamburger menu

## Skill: Retry Incorporation
When given evaluator feedback, read every point and address it specifically. Do not regenerate from scratch — improve the existing structure based on the feedback.
